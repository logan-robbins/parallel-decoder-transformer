\section{Related Work}

We organize prior work by the \textbf{shape of the output interface a single model exposes}. On that axis, the existing landscape populates three points; PDT introduces a fourth.

\subsection{One stream, one next token}

The default output interface of a causal language model is a single stream emitting $p(x_t \mid x_{<t})$. PDT retains the single-model setting and the frozen pre-trained decoder but replaces the single-stream assumption with $K$ coexisting causal streams sharing a latent workspace.

\subsection{One stream plus lookahead verification}

Speculative Decoding~\cite{leviathan2023fast}, Blockwise Parallel Decoding~\cite{Stern2018}, Medusa~\cite{cai2024medusa}, Hydra~\cite{Ankner2024Hydra}, EAGLE~\cite{EAGLE2024}, SpecDec++~\cite{Huang2024SpecDecPP}, Lookahead Decoding~\cite{Li2024lookahead}, Jacobi parallel decoding~\cite{Santilli2023Jacobi}, and CLLMs~\cite{Kou2024CLLM} keep a single committed causal stream and add drafted future tokens accepted or rejected by the same model \cite{Yan2024,survey2025}. Even a perfectly accepted Medusa or EAGLE draft leaves a single output sequence. PDT's $K$ threads persist and coordinate across synchronized rounds.

\subsection{One stream plus parallel positions}

Non-Autoregressive Transformers~\cite{Gu2018NAT}, Mask-Predict~\cite{Ghazvininejad2019MaskPredict}, the Glancing Transformer~\cite{Qian2021GLAT}, and diffusion-based language models~\cite{Li2022DiffusionLM,Lou2024SEDD} parallelize \emph{positions} within one output. The output interface remains a single sequence.

\subsection{$K$ causal streams sharing a latent workspace}

The closest points to PDT are \emph{Hogwild!}~Inference~\cite{Rodionov2025}---concurrent generations of one model attending to each other's KV caches---and SPRINT~\cite{Xiao2025sprint}, which interleaves a planner and a parallel executor within a single reasoning model. Conceptual precedents include shared neural workspaces~\cite{Goyal2021} and the Consciousness Prior~\cite{Bengio2017ConsciousnessPrior}.

PDT adds three properties that prior methods on this axis do not combine: a \emph{mandatory prompt-time latent planner} that seeds a shared snapshot 0 \emph{per stream} before any token is emitted; an \emph{embeddings-only Dynamic Notes Bus} with a reveal delay $\Delta$; and an \emph{agreement-gated commit protocol} that decides per block whether the parallel frontier may advance.

\subsection{External multi-call orchestration}

Skeleton-of-Thought~\cite{ning2023skeleton}, PSLM~\cite{PSLM2024}, GPTSwarm~\cite{Zhuge2024GPTSwarm}, Mixture-of-Agents~\cite{Wang2025MoA}, blackboard-style multi-agent systems~\cite{Han2025Blackboard,Salemi2025Blackboard}, topology-adaptive AMAS~\cite{Leong2025AMAS}, and LatentMAS~\cite{Zou2025LatentMAS} compose multiple calls externally. PDT modifies the output interface of a \emph{single} model so that coordination is a latent workspace \emph{inside} the decoder.
