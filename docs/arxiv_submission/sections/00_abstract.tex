\begin{abstract}
Autoregressive decoding in large language models remains sequential even when a task naturally decomposes into parallel sections. External decomposition methods can launch multiple prompts concurrently, but they create no shared model-internal state to keep those sections non-overlapping and coherent. We present the \textbf{Parallel Decoder Transformer (PDT)}, a frozen-trunk architecture for model-internal parallel decoding. Before any stream emits tokens, a mandatory prompt-time planner predicts fixed latent plan slots over a shared vocabulary and projects the resulting embeddings as snapshot~0 on an embeddings-only Dynamic Notes Bus. During decoding, each stream reads lagged note embeddings through Speculative Note Conditioning (SNC) cross-attention at every step, while coverage and agreement heads regulate plan-item ownership and trigger rollback on coherence failure. The entire coordination stack is trained end-to-end via knowledge distillation on a frozen backbone without modifying pre-trained weights.
\end{abstract}
